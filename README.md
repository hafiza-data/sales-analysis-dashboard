# Sales Analysis Dashboard

## About the Project
In this project, I worked on analyzing international sales data using Power BI to understand overall business performance. The dataset contains around two years of sales data with more than 500 records.

## Tools Used
Power BI (Data Modeling, DAX, Visualization)

## What I Did
- Designed a star schema data model for better data organization
- Created key measures like Total Sales, Total Orders, Average Order Value, and Year-over-Year Growth using DAX
- Built multiple report pages for different types of analysis
- Used slicers, drill-through, and tooltip features to improve interactivity
- Created a separate validation page to verify totals and ensure data accuracy
